# Mapping2
Create mid-between interpoled measures CSV file of mapping_corrections.csv

Console application to create CSV file based on original mapping_corrections.csv.

The original file must be in the same directory as this utility.

The resulting file will be mapping_corrections-2.csv.

This file contains all interpolated mid points as following:

<pre>
O X O
X X X
O X O
</pre>

O = original measure

X = new interpolated measure
